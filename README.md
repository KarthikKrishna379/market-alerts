# Market Alerts

Automated stock market alerts for NIFTY indices with GitHub Actions and Telegram bot integration.

## Features

- Monitor NIFTY 50, NIFTY Next 50, and NIFTY Midcap 150 indices
- Calculate technical indicators:
  - 200-Day Moving Average (200DMA)
  - 50-Day Moving Average (50DMA)
  - 52-Week High
  - % difference from 52-Week High
- Send alerts via Telegram bot
- Scheduled execution via GitHub Actions
